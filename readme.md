# 荧光颗粒检测器 v3.2

一个基于PySide6的图形界面工具，用于自动检测和统计荧光显微镜图像中的颗粒。

## 功能特点

- **拖拽加载**：支持单张或批量图片拖拽
- **双检测算法**：
  - LoG斑点检测：提供颗粒半径信息
  - 局部最大值检测：速度更快
- **多种检测模式**：平衡、高精度、高召回
- **批量处理**：自动处理文件夹内所有图片
- **结果导出**：支持保存结果图和CSV坐标

## 安装说明

### 1. 安装Python
确保已安装Python 3.10或更高版本。

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

或者手动安装：

```bash
pip install PySide6 opencv-python numpy scipy scikit-image scikit-learn pandas matplotlib
```

3. 运行程序

```bash
python particle_detector.py
```

使用说明

加载图片
- 拖拽：将图片或文件夹拖到左侧虚线框
- 点击：点击虚线框选择文件

设置参数

参数	说明	
颗粒颜色	选择红色、绿色或蓝色通道	
检测模式	平衡/高精度/高召回	
颗粒大小均匀	勾选如果所有颗粒大小相似	
检测方法	至少选择LoG或局部最大值一项	
批量处理	勾选以自动保存所有结果	

检测模式说明
- 平衡：默认参数，适合大多数情况
- 高精度：减少误检，适合质量要求高的分析
- 高召回：检测更多颗粒，可能包含少量噪点

查看结果
- 右侧显示三栏对比：原始图、LoG结果、局部最大值结果
- 统计信息面板显示颗粒数量和推荐计数

导出结果
- 保存结果图：保存高分辨率对比图
- 导出CSV坐标：导出所有颗粒的坐标和半径

批量处理

1. 勾选"批量处理（自动保存结果到output文件夹）"
2. 选择或拖拽包含图片的文件夹
3. 点击"开始检测"
4. 选择输出文件夹
5. 程序自动保存所有结果图和汇总CSV

文件说明

文件	说明	
`particle_detector.py`	主程序	
`requirements.txt`	依赖列表	
`README.md`	使用说明	

常见问题

Q: 检测到的颗粒数量不准确？

A: 尝试调整检测模式或勾选/取消"颗粒大小均匀"

Q: 批量处理时程序卡住？

A: 图片数量过多时请耐心等待，每10张会更新进度

Q: 某些图片无法加载？

A: 确保图片格式为PNG、JPG、TIF或BMP

技术说明

- 使用OpenCV进行图像处理和标记绘制
- 使用scikit-image进行斑点检测
- 使用PySide6构建图形界面
- 使用matplotlib保存高质量结果图
